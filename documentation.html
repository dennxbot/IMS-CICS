üìä System Overview

The Internship Management System (IMS-CICS) is a comprehensive web-based platform designed to streamline the entire internship process for students, academic coordinators, and host companies at Cagayan State University.

üéØ Key Objectives

Centralized Management

Single platform for all internship-related activities

Real-time Tracking

Monitor internship hours and progress in real-time

Automated Reporting

Generate and submit reports digitally

Compliance Monitoring

Ensure students meet internship requirements

Communication Hub

Facilitate coordination between all stakeholders

Component Technology

Backend Next.js API Routes, Server Actions

UI Components shadcn/ui
Deployment Vercel

üöÄ Getting Started

System Requirements

  • Modern web browser (Chrome, Firefox, Safari, Edge)
  • Stable internet connection
  • Valid email address (for registration)

Registration Process

For Students:

Click "Get Started" or "Register" on the landing page
Fill in the registration form with your details (Student ID, Full Name, Email, Contact Number, Address, Course, Company, Password)
Click "Create Account"
Check your email for a confirmation link
Click the confirmation link to verify your account
Login with your credentials
üìù Note for Administrators: Admin accounts are created by system administrators. Contact the IT department for account creation.
